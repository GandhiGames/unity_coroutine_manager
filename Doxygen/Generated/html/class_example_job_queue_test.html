Detailed Description

Includes a number of methods to test the functionality of the CM_JobQueue class. Each method showcases a particular functionality of the CM_JobQueue that you can implement in your own projects. Each method returns an ienumerator so that it can added to a seperate job queue to be run in sequence for test purposes.
